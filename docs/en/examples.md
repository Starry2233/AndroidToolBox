# AndroidToolBox Usage

Download latest AndroidToolBox 7z/zip from [this link](https://atb.xgj.qzz.io/). Unzip it and run `双击运行.exe` with Administrators Privilege.

::: details
Only AndroidToolBox on Windows 10 and 11 has been tested. NOT SUPPORT Windows 7.
:::

For more information, please watch the Bilibili Videos.
